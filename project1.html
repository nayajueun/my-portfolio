HR & Manpower Analytics Web Application Project

Overview:

The company expressed a pressing need to revolutionize their HR and manpower analytics. Initially, an executable was developed using Python's Tkinter library to meet these requirements. The success and proficiency of this solution brought forth an additional request: a more flexible, user-friendly web application to further optimize their data entry and analytics processes.

Project Objectives:

  • Transition from a desktop executable to a fully functional web application.
  • Implement a digital logging system, eliminating manual data entry and subsequently reducing labor costs.
  • Ensure the application is intuitive and user-friendly, catering to individuals without a technical background.
  • Optimize for multiple devices, including desktop and iPad, to maximize accessibility and ease of use.

Technical Implementation:

  • Platform & Backend Framework: Flask was chosen for its versatility, ease of use, and compatibility with various extensions.
  • User Authentication: A user authentication system was built to differentiate between worker and admin roles, ensuring data security and role-specific functionalities.
  • Data Management: PostgreSQL, in combination with SQLAlchemy. This decision hinged on PostgreSQL's enhanced scalability and robustness, making it ideal for the deployment scenario in this web application.
  • Data Analytics Tool: For robust data analysis, data from SQLAlchemy was seamlessly converted to pandas dataframes. This allowed for more advanced data manipulations and insights to be derived from the stored information.
  • User Interface & Experience: Leveraging Bootstrap, the application was designed to be mobile-responsive, ensuring a consistent experience across devices. This consideration was crucial as supervisors were expected to access the application via iPads.
  • Testing: Comprehensive multi-device testing was conducted to assure functionality and optimize the user experience.

Outcome & Feedback:

The application was deployed to an external server and functions with user authentication. It improved data logging and analytics efficiency while significantly cutting labor costs by automating formerly manual data analysis for quotations and contract creations. The successful launch of both the executable and the web application deepened our relationship with the company, evident from their positive feedback and ongoing collaboration requests.
